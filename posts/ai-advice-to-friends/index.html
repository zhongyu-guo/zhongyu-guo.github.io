1. 先说怎么用#

1.1 怎么用#

  1. 国内模型可以直接使用产品

  2. 一些强大的国外不是很方便用到的闭源模型（首先一定要注意信息安全）

    • 通过中转平台，他们通过各种技术手段（比如说反代之类的），把各个家模型的 API 拿过来
    • 结合 Cherry Studio、Chatbox AI 这些工具，可以很方便地使用这些 API 进行对话
    • 以不高的成本，可以很方便且快速地用到最新的模型

1.2 从信息输入，打破信息茧房维度#

  • Gemini、Perplexity、Deep Research 模式
  • 面对纷繁复杂的爆炸信息，AI 寻源、AI 总结，提升高质量信息输入效率
  • 比如，RSS 订阅几个优秀的长文来源

1.3 作为一个高智商的决策参考幕僚#

  • GPT-5.2-Pro
  • 不管是人生规划还是业务决策，因为它综合了很多案例历史数据，懂得结构化分析
  • 比如让它给个建议，述职报告还可以从哪些维度补充；让它反驳你的观点，看看是不是有什么你没有考虑到的漏洞

1.4 得力的操作助手#

  • OpenClaw、AionUI
  • 原本主要用在 Vibe Coding 里面编辑文件内容以及操作各个文件的能力，现在逐步被整合进个人日常工作助手，作为 Agent 来使用。不管是移动整合文件、比较文档内容，还是直接帮你创作内容，现在的工具都可以帮你很快捷地实现。你只需要通过对话去命令它就行了。

1.5 高情商的聊天对象#

  • 规模大一点的模型都可以
  • 不管是职场、家庭，还是朋友之间，在各种人际关系场景的处理上，你都可以通过对话向它获取建议
  • 基本上，它已经相当于是一个有着丰富处理经验的专家，特别是它拥有大量人际关系科学的知识积累，可以给你提供一些不错的建议

2. AI是挑战还是机遇？#

看完前面已经有那么多有益的用处了，那结论也呼之欲出：机遇大概率是远远大于危机的。

对于企业而言#

员工的生产力应该通过不高的工具付费，实现大幅的增长。除了在一些与 AI 能力高度重合且没有新需求需要满足的场景中可能需要裁员，在其他情况下，完全可以将工具能力提升所带来的人力冗余，投入到扩展更大的业务中，直接扩大产能。这样一来，在竞争中不仅能把成本打下来，还可以借此获得更大的优势。

对于个人而言#

实际上一个人的决策能力、认知能力以及产出能力，都会因为 AI 工具的使用而得到极大提升。如果一个人比之前更加胜任一个岗位，而整个社会又在提高平均水平，其实大家的相对水平并没有太大的变化，所以应该也不会特别影响到职位。

对于社会而言#

AI 对这个社会的需求带来的变化应该也是扩大的。随着 Vibe Coding 以及 Agent 这种工具的出现，已经可以发现，人的闲暇时间正在变得越来越多。满足人的基本生存所需要的时间越来越短，更多的时间被用来获得精神上的满足。为了获得精神上的满足，会需要新的供应，比如剧集之类的内容，或者是一些户外运动等各种形式。

4. 我们应该怎么看待 AI？#

随着像 ModeBot、OpenCloud 这样工具的出现，它们在 MacBook 里面已经体现出一种组织形式。你已经很难把它们与一个独立的人去进行区别了，我们应该怎么去看待它们呢？

我从现在就开始把它当成了我的伙伴。 我已经开始准备让它拥有长期记忆，它既是我的数字化身，也是数字世界里的一个独立人格。

AI 时代已经来临，与其担忧，不如拥抱。学会使用工具，让 AI 成为你的助力而非威胁。
